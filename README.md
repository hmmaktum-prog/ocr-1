# PDF থেকে DOCX রূপান্তরণ - PaddleOCR-VL মোবাইল অ্যাপ

একটি সম্পূর্ণ অফলাইন অ্যান্ড্রয়েড অ্যাপ্লিকেশন যা PaddleOCR-VL-1.5 ব্যবহার করে PDF ফাইল থেকে সরাসরি DOCX (Microsoft Word) ফাইলে রূপান্তর করে।

## বৈশিষ্ট্য

- **অফলাইন প্রক্রিয়াকরণ**: সম্পূর্ণভাবে ডিভাইসে চলে, ইন্টারনেটের প্রয়োজন নেই
- **পৃষ্ঠা-দর-পৃষ্ঠা রূপান্তর**: প্রতিটি পৃষ্ঠা স্বতন্ত্রভাবে প্রক্রিয়া করা হয়
- **PaddleOCR-VL-1.5 ইঞ্জিন**: শক্তিশালী অপ্টিক্যাল ক্যারেক্টার রিকগনিশন
- **বাংলা ভাষা সমর্থন**: বাংলা পাঠ্য সম্পূর্ণভাবে স্বীকৃত
- **DOCX আউটপুট**: Microsoft Word সম্বন্ধযুক্ত ফরম্যাটে স্বয়ংক্রিয় সংরক্ষণ
- **ব্যবহারকারী-বান্ধব ইন্টারফেস**: Kivy ফ্রেমওয়ার্কের সাথে সুন্দর মোবাইল UI

## প্রযুক্তি স্ট্যাক

- **Python 3**: মূল প্রোগ্রামিং ভাষা
- **Kivy**: মোবাইল অ্যাপ্লিকেশন ফ্রেমওয়ার্ক
- **PaddleOCR**: অপ্টিক্যাল ক্যারেক্টার রিকগনিশন ইঞ্জিন
- **python-docx**: DOCX ফাইল তৈরি এবং সম্পাদনা
- **pdf2image**: PDF থেকে ইমেজ রূপান্তরণ
- **PyMuPDF**: PDF প্রক্রিয়াকরণ
- **Buildozer**: APK নির্মাণ টুল

## প্রয়োজনীয়তা

### ডেভেলপমেন্ট পরিবেশের জন্য:
- Python 3.8 বা উচ্চতর
- Buildozer (Linux/macOS এ)
- Java Development Kit (JDK)
- Android SDK

### ডিভাইসের জন্য:
- Android 7.0 (API 24) বা উচ্চতর
- কমপক্ষে 500MB ফ্রি স্টোরেজ (OCR মডেলের জন্য)
- 2GB RAM (সুপারিশকৃত)

## ইনস্টলেশন এবং বিল্ড

### ধাপ 1: ডেপেন্ডেন্সি ইনস্টল করুন

```bash
pip install -r requirements.txt
```

### ধাপ 2: APK বিল্ড করুন

```bash
# প্রথমবার (দীর্ঘ সময় নেবে - 30-60 মিনিট)
buildozer android debug

# সম্পূর্ণ রিলিজ বিল্ড
buildozer android release
```

### ধাপ 3: ডিভাইসে ইনস্টল করুন

```bash
# USB ডিবাগিং সক্ষম করে ডিভাইস সংযুক্ত করুন
adb install -r bin/pdftoDocxocr-1.0-debug.apk

# বা সরাসরি APK ফাইল ট্রান্সফার করে ইনস্টল করুন
```

## OCR মডেল তথ্য

অ্যাপ্লিকেশন প্রথম চালু করার সময়:
- PaddleOCR স্বয়ংক্রিয়ভাবে প্রয়োজনীয় মডেল ডাউনলোড করবে (~300MB)
- মডেল ফাইলগুলি ডিভাইসে ক্যাশ করা হয় পরবর্তী ব্যবহারের জন্য
- প্রথম রূপান্তর সবচেয়ে বেশি সময় নেবে (মডেল লোডিংয়ের কারণে)

## পারফরম্যান্স টিপস

1. **নেটিভ ডিভাইসে চালান**: এমুলেটরে ধীর হতে পারে
2. **বড় PDF এড়িয়ে চলুন**: 100+ পৃষ্ঠার PDF রূপান্তর করা সময় সাপেক্ষ
3. **উচ্চ রেজোলিউশন পিডিএফ**: 150-200 DPI সর্বোত্তম ফলাফলের জন্য
4. **যথেষ্ট স্টোরেজ নিশ্চিত করুন**: রূপান্তরের সময় অস্থায়ী ফাইল তৈরি হয়
